Fantasy Goal

  • Industry: iGaming, Fantasy Football
  • Website: https://fantasy-goal.com/
Fantasy Football Betting Management System deployed in South American betting offices. Here, we applied Microservices and Serverless Architecture, utilizing React, and Azure Kubernetes Service, while optimizing ticket archiving with CosmosDb and orchestrating critical calculations and data fetching through Azure Functions

Project Requirements

  • .NET
  • React
  • Microsoft Azure
  • Durable Functions
  • Service Bus
  • Serverless Architecture
  • SignalR
  • CosmosDb
  • Kubernetes

Project Overview

Project serves React on frontend applications and serving Microservices on backend are orchestrated on Azure Kubernetes Service (AKS). Projects work with data vendors such as live Football feeds and this data constantly gets fetched/updated from vendor Apis. These processes are orchestrated with Durable Functions on backend. Any live information has been broadcast to users with Azure SignalR Service. Project relies on massive amounts of tickets so frequent and active tickets are managed on SQL Server and archived ones are stored on CosmosDb(NOSQL) as readonly
